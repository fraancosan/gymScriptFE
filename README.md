## Pasos para Ejecutar Localmente

- Instalar Angular: npm install -g @angular/cli@latest
- Instalar Dependencias: npm install
- Ejecutar Servidor
  - En consola escribir: ng serve
  - El servidor se creara en: http://localhost:4200/
